The Story So Far

Documentary Editor, Developer, AI Creator

I am Vinayak, a creative generalist working with Tejus Creative in Mohali, Punjab. I focus on structural documentary edit pacing, lightweight interactive coding, and automated LLM prompt pipelines.

My creative journey began behind the timeline in video editing, learning how visual structure, pacing cuts, and sound layering establish emotional narrative truths. As my work expanded, I developed a passion for writing clean frontend code to build high-performance web showcases, and integrating generative AI tools to automate storyboard processes.

I believe that tools should serve the content—whether it is raw film logs, semantic HTML code, or LLM-driven pipelines. By keeping file structures lean and removing bloat, I focus on delivering speed and clean visual outcomes.

"A story should be as complex as necessary, but the system delivering it should be as simple and lightweight as possible."

Core Creative Philosophy

How my multi-disciplinary background translates into project outcomes.

Pacing is Everything

Whether editing a documentary sequence or developing web animations, timing defines the user experience. I build structural pacing into cuts and transitions to command attention naturally.

Lean & Clean

Bloated libraries slow down load times and make timelines chaotic. I write vanilla code and construct structured timelines to keep systems maintainable and lightweight.

Content-First Automation

I view AI as a production accelerant, not a replacement. I use generative script integrations to automate drafting storyboards, leaving more room to focus on high-fidelity creative outputs.
